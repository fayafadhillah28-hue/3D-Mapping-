# Payung Teduh: Perlindungan dalam Cahaya ☂️💡

Sebuah *Landing Page* portofolio multi-halaman untuk proyek *3D Video Mapping* "Payung Teduh". Proyek ini menggabungkan seni proyeksi cahaya dengan medium fisik (payung) untuk menceritakan kisah tentang interaksi air, cahaya, dan perlindungan.

## 🔥 Fitur Utama
- **Desain Terang & Minimalis**: Estetika modern nan elegan yang terinspirasi dari gaya majalah digital premium dengan paduan warna dominan putih, abu muda, dan sentuhan warna *peach/salmon* pastel.
- **Hero Split Screen**: Halaman utama (*Beranda*) menggunakan tata letak layar terbagi secara asimetris dengan panel teks gelap di sebelah kiri dan *video loop* dinamis di sisi kanan.
- **Bento Grid Layout**: Menggunakan tata letak struktur kotak modern untuk galeri pemetaan karya Resolume dengan bayangan lembut berdimensi.
- **Multi-Page Architecture**: Dibangun dengan struktur navigasi yang modular dan ringan dengan 3 pilar halaman terpisah untuk SEO dan keterbacaan yang optimal.
- **Responsif**: Tetap indah di berbagai tipe panel (Mobile, Tablet, Desktop).

## 🛠️ Cara Mengakses Web
Situs ini merupakan **Static Web Application** murni yang dibangun dengan HTML5 dan CSS Vanilla tanpa tambahan dependensi framework apa pun. Oleh karena itu, cara menjalankannya sangatlah mudah:

1. Ekstrak atau muat folder `thr_portofolio/` dari lokal disk Anda.
2. Cukup klik ganda (atau *drag-and-drop*) file **`index.html`** ke penjelajah web (*browser*) modern Anda seperti Google Chrome, Safari, atau Firefox.
3. Anda dapat menelusuri halaman `Beranda`, `Filosofi`, maupun `Galeri` menggunakan bilah navigasi di menu atas halaman web.

## 🎨 Teknologi yang digunakan
- **HTML5**: Semantik dasar penataan landing page.
- **CSS Vanilla (Native)**: CSS *Grid*, *Flexbox*, *Media Query*, dan variabel *CSS root* kustom untuk sistem tema cerah.
- **Intersection Observer / CSS Animations**: Transisi elemen efek transparan (*Fade-in*) saat scroll.
